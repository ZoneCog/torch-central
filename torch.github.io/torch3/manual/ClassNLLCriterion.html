Documentation

This criterion can be used to train *in classification* a GradientMachine object using the StochasticGradient trainer. It then maximizes the log likelihood of the data.

If we write for the output of the GradientMachine, it supposes that

  • the outputs are log-probabilities.
  • is the probability for the class
  • the predicted class follows a multinomial distribution with parameters

The number of target frames in DataSet must correspond to the number of input frames given to this criterion.
