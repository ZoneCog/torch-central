class LMInteriorLevelWordEntry

This class is used internally within the language model n-gram data structures.

Documentation

This class is used internally within the language model n-gram data structures. It contains a list of LM probabilities relevant to the level of the tree structure where it exists, and a link to the next level of the tree structure.

This class has no child classes.
